## Slim ZSH

https://github.com/changs/slimzsh

```
git clone --recursive https://github.com/changs/slimzsh.git ~/.slimzsh
source "$HOME/.slimzsh/slim.zsh"

touch ~/.zshrc

echo 'source "$HOME/.slimzsh/slim.zsh"' >> ~/.zshrc
```
